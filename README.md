# Expense Tracker Dashboard

A modern and responsive Budget Management Web Application built using React.js and Tailwind CSS. This application helps users track their income, expenses, and overall financial balance with an interactive dashboard and analytics.

## Features

- Add Income & Expenses
- Real-time Balance Calculation
- Expense Categories
- Transaction History
- Delete Transactions
- Responsive Dashboard UI
- Local Storage Persistence
- Charts & Analytics
- Mobile Friendly Design

---

## Tech Stack

- React.js
- Tailwind CSS
- Recharts
- React Icons
- Local Storage API
